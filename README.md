# 🏥 Hospital Management System

## 🚀 Features
- User Authentication (Patient & Doctor)
- Doctor Listing (Dynamic from DB)
- Book Appointment
- Cancel Appointment
- Doctor Dashboard (View Patients)

## 🛠️ Tech Stack
- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB

## ⚙️ Setup

### Backend
cd Backend
npm install
create .env file:
MONGO_URI=your_mongo_url
PORT=5000

npm start

### Frontend
cd frontend
npm install
npm start
